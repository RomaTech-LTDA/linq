@romatech/linq

GitHub npm

A lightweight library that extends JavaScript arrays with LINQ-style methods inspired by .NET's System.Linq. Introduces a LazyQuery pipeline with deferred execution — the same model as IEnumerable<T> in C#.

Installation

npm install @romatech/linq

Quick Start

import '@romatech/linq';

const users = [
  { name: 'Alice', age: 30, active: true },
  { name: 'Bob',   age: 25, active: false },
  { name: 'Carol', age: 30, active: true },
  { name: 'Dave',  age: 22, active: true },
];

// Exactly like C#: users.Where(...).OrderBy(...).Select(...).Take(...).ToList()
const result = users
  .where(u => u.active)
  .orderBy(u => u.age)
  .select(u => u.name)
  .take(2)
  .toArray();

console.log(result); // ['Dave', 'Alice']

Features

Lazy Evaluation

Nothing executes until a terminal operator is called — single-pass iteration, no intermediate arrays.

40+ Operators

where, select, selectMany, take, skip, distinct, union, intersect, except, groupBy, innerJoin, and more.

Short-circuit

first(), any(), all(), and take() stop iteration as soon as the result is determined.

Full TypeScript Support

Typed LazyQuery<T> pipeline and OrderedArray<T> interface included.

Sorting Chain

orderBy(), orderByDesc(), thenBy(), thenByDesc() — Schwartzian transform (O(n) key extraction).

Zero Dependencies

Lightweight prototype extension. Works with CommonJS, ESM, and bundlers. Node.js ≥ 11.
